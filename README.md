# Relatório Técnico – Aula de Frameworks e Bootstrap 5

## 1. Contextualização

Durante a aula do curso de **Análise e Desenvolvimento de Sistemas (ADS)** da **Unicesumar**, ministrada pelo professor Leonardo Rocha (inserir link oficial), foi realizada uma introdução prática aos conceitos de Framework, utilização do Bootstrap 5 e versionamento de código com Git.

---

## 2. Conteúdo Abordado

### 2.1 Framework

Foi apresentado o conceito de framework como uma estrutura de desenvolvimento que fornece componentes reutilizáveis, padronização de código e aceleração no processo de construção de aplicações.

### 2.2 Bootstrap 5

Foi realizada a exploração do framework front-end Bootstrap 5, com foco em:

- Estruturação de layout
- Sistema de grid
- Componentes prontos
- Estilização responsiva
- Classes utilitárias

### 2.3 Git

Foram aplicados conceitos básicos de versionamento utilizando Git, incluindo:

- Inicialização de repositório
- Adição de arquivos ao staging
- Realização de commits
- Controle de alterações no projeto

---

## 3. Ferramentas Utilizadas

- Visual Studio Code (VSCode) como ambiente de desenvolvimento
- Git para controle de versão
- Bootstrap 5 como framework CSS
- HTML5 para estruturação do template

---

## 4. Atividade Prática

Foi utilizado como base o template oficial "Sign-in" disponibilizado pelo Bootstrap, que consiste em um modelo de página de autenticação (login) estruturado em HTML.

A partir do template original, foram realizadas as seguintes modificações:

- Tradução e adaptação do conteúdo textual para o português
- Inclusão de seção FAQ (Frequently Asked Questions)
- Ajustes estruturais no HTML para adequação ao contexto da atividade
- Personalizações pontuais na organização dos elementos

---

## 5. Objetivos de Aprendizagem

- Compreender o papel de frameworks no desenvolvimento web
- Aplicar o Bootstrap 5 na construção de interfaces responsivas
- Manipular templates pré-existentes para personalização
- Utilizar Git como ferramenta de versionamento e controle de alterações
